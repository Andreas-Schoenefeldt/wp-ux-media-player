# wp-ux-media-player
Improves the standard wordpress media player to have a better skin and usability.

## Usage:

Via shortcode:

```html
[ux-audio-player media-ids="id1,id2,id3" title="My Player Title" collapse-from="4"] 
```

* media-ids: required comma seperated list of media ids
* title: optional title for the player
* collapse_from: optional number of tracks, that are shown directly without collapsing
